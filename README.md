# Getting started

## Install

```
npm install --save reelm
```

## Building examples from sources

```
git clone https://github.com/tihonove/reelm.git
cd reelm
npm install
```

### Counter example

```
cd 01_counter
npm install
npm start
```

### Pair of counters example

```
cd 02_pair_of_counters
npm install
npm start
```
